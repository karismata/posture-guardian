# Posture Guardian (자세 지킴이)

이 앱은 사용자 컴퓨터의 웹캠을 사용하여 실시간으로 올바른 자세를 모니터링합니다.

## 🚀 배포 및 업로드 방법

이 폴더에 있는 **두 개의 파일**만 있으면 웹사이트를 운영할 수 있습니다.

1.  **`index.html`** (웹사이트 화면)
2.  **`app.js`** (자세 분석 로직)

### ✅ 필수 조건: HTTPS (보안 연결)
**중요합니다!**
웹 브라우저는 보안상의 이유로 `http://` (보안되지 않은 사이트)에서는 카메라 사용을 차단합니다.
따라서 반드시 **`https://` (자물쇠 아이콘이 뜨는 사이트)** 로 배포해야 카메라가 작동합니다.

### 추천 무료 호스팅 서비스
다음과 같은 서비스에 이 두 파일을 그대로 업로드하면 바로 사용 가능합니다.
*   **Netlify Drop**: 회원가입 후 파일을 드래그앤드롭만 하면 주소가 생성됩니다. (추천)
*   GitHub Pages
*   Vercel

### 로컬 실행 (내 컴퓨터에서만 쓸 때)
인터넷 없이 혼자 쓰고 싶다면 지금처럼 `python` 서버를 켜거나, VS Code의 'Live Server' 확장 기능을 사용하면 됩니다.
단순히 파일을 더블클릭해서 열면 보안 정책 때문에 카메라가 작동하지 않을 수 있습니다.
